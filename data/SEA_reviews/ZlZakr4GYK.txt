 **Summary:**
The paper introduces a novel method for modeling non-uniform event sequences using a continuous convolutional neural network (COTIC). This model is designed to process event sequences with non-uniform time intervals, which is a significant challenge in modeling temporal point processes. The COTIC model employs a multi-layer convolutional architecture to capture complex dependencies and a multi-layer perceptron for intensity prediction. The paper presents a comprehensive evaluation of the model on eight datasets, showing that it outperforms existing models in predicting event types and return times. The methodology is supported by a thorough theoretical analysis and empirical results, demonstrating the model's superior performance and efficiency.
